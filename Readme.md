# SIC AI Project

This repository contains code, notebooks, and resources for the SIC AI project, including assignments, capstone projects, labs, and various machine learning experiments.

## Structure
- **Assignment 01/**: Contains assignment files and related resources.
- **Capstone Proj/**: Includes notebooks and models for capstone work, such as EDA, pretrained and finetuned models, and image classification tasks.
- **Class TASK/**: Organized tasks for classwork.
- **Lab X/**: Lab folders with Jupyter notebooks and datasets for different topics (e.g., MNIST, FashionMNIST, Titanic, Breast Cancer, Iris, etc.).
- **qmnist/**: QMNIST dataset and related files.
- **slides/**: Presentation materials.

## Notable Files
- Jupyter notebooks for data analysis, model training, and experimentation.
- Pretrained and finetuned PyTorch models (`*.pth`, `*.pt`).
- CSV datasets for various machine learning tasks.

## Getting Started
1. Clone the repository:
   ```zsh
  [ git clone https://github.com/hmzi67/music.git](https://github.com/hmzi67/sci-ai-course)
   ```
2. Install required Python packages (see individual notebooks for dependencies).
3. Open notebooks in Jupyter or VS Code and follow instructions within each notebook.
